1. Panoramica Generale

Idro-Dashboard è una suite completa in Python per l'estrazione automatizzata dei dati dagli Annali Idrologici (ARPAE/Servizio Idrografico dell'Emilia-Romagna) e una Dashboard Web Interattiva per l'analisi statistica avanzata delle precipitazioni massime e la costruzione delle Curve di Possibilità Pluviometrica (CPP).

🎯 Pubblico Target

  • Ingegneri idraulici e ambientali
  • Geologi tecnici e ambientali
  • Tecnici della Protezione Civile
  • Studenti di ingegneria civile e ambientale

🏗️ Ambito di Applicazione

  • Progettazione idraulica (acciaio, tombinature)
  • Verifica ordinarie e straordinarie
  • Stima portate di piena (metodo razionale)
  • Analisi climatica e trend storici

2. Funzionalità Principali

L'applicazione si compose di due moduli fondamentali:

📂 Modulo di Estrazione Dati

  • Lettura automatica dei PDF degli Annali Idrologici pubblicati da ARPAE
  • Analisi testuale nativa per PDF vettoriali (es. 2024) - utilizzo di pdfplumber
  • (Opzionale) Motore OCR per le scansioni storiche (1916-1923) - supporto Tesseract
  • Raggruppamento e pulizia dei dati con esportazione in Excel

🖥️ Modulo Dashboard (Streamlit)

  • Mappa Interattiva: localizzazione geografica delle stazioni pluviometriche
  • Calcolo CPP: interpolazione dei parametri a e n per vari Tempi di Ritorno
  • Modelli Statistici: scelta dinamica tra Distribuzione di Gumbel (EVI) e GEV
  • Intervalli di Confidenza: visualizzazione dell'incertezza statistica (Fasce al 95%)
  • Trend Climatici: analisi grafica delle serie storiche
  • Esportazione: download CSV parametrico per HEC-RAS, SWMM
  • Sezione Idraulica: calcolo tirante con formula di Strickler

6. Fondamenti Teorici

6.1 Curva di Possibilità Pluviometrica (CPP)

Equazione del monomio
h(t) = a · tn

dove:

  • h = altezza di precipitazione [mm]
  • t = durata della precipitazione [ore]
  • a = coefficiente dipendente dal Tempo di Ritorno [mm/hn]
  • n = esponente caratteristico del luogo (0.30 < n < 0.50 per Italia)

6.2 Distribuzione di Gumbel (EVI)

Variable ridotta
yTr = -ln[ -ln(1 - 1/Tr) ]
Precipitazione di progetto
hTr = u + yTr / α

La stima dei parametri u (moda) e α (scala) avviene tramite il metodo dei momenti:

Parametri stimati
α = π / (σ · √6)     u = μ - (0.5772 / α)

Con μ = media campionaria, σ = deviazione standard campionaria.

6.3 Distribuzione GEV (Generalized Extreme Value)

Funzione di distribuzione
H(x) = exp{ -[ 1 + ξ(x-μ)/σ ]-1/ξ }

dove μ = parametro di posizione, σ = parametro di scala, ξ = parametro di forma (shape). Il fitting avviene per maxima likelihood estimation (MLE).

6.4 Calcolo Idraulico (Tirante con Strickler)

Formula di Strickler
Q = Ks · A · R2/3 · i1/2
Geometria sezione trapezoide
A = h · (b + z·h)     P = b + 2h√(1+z²)     R = A/P

7. Diagramma CPP e Rappresentazione Grafica

Curva di Possibilità Pluviometrica - Rappresentazione schematica

💡 Come leggere il grafico

  • Asse X: durate standard (1h - 24h)
  • Asse Y: altezze di precipitazione [mm]
  • Curve: una per ogni Tr desiderato
  • Area ombreggiata: intervallo di confidenza (95%)
  • Punti: precipitazioni osservate nel database

📐 Linearizzazione

Attivando la scala Log-Log le curve CPP diventano lineari:

Forma linearizzata
log h = log a + n · log t

L'interpolazione lineare (OLS) su log-log fornisce i parametri a = exp(intercept), n = slope.

8. Esempio di Utilizzo

Caso di studio: Stazione di Bologna

Dati di input

  • Stazione: Bologna
  • Tempo di Ritorno: Tr = 50 anni
  • Modello: Gumbel (EVI)
  • Serie storica: 45 anni (1979-2023)

Risultati ottenuti

  • a = 35.82 mm/hn
  • n = 0.382
  • R² = 0.9824
  • Tr = 50 anni

Verifica rapida

Per una pioggia di durata t = 6 ore e Tr = 50 anni, la precipitazione di progetto:

Calcolo diretto
h = 35.82 · 60.382 = 35.82 · 2.24 ≈ 80.2 mm
